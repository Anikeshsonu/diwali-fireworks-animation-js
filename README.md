# 🎆 Diwali Fireworks Animation JS

![JavaScript](https://img.shields.io/badge/JavaScript-Animation-yellow)
![License](https://img.shields.io/badge/License-MIT-green)
![Festival](https://img.shields.io/badge/Festival-Diwali-orange)

A **lightweight centralized JavaScript fireworks animation** designed for **Diwali celebration websites**. 

This project provides a **single reusable JS file** that can be integrated into **any website** to display beautiful **Diwali fireworks effects** instantly.

---

## ✨ Features

*   **Centralized reusable JS animation**: Just import and it works.
*   **Lightweight and fast**: No external dependencies or heavy assets.
*   **Easy plug-and-play integration**: Works with any HTML website.
*   **Mobile friendly**: Responsive canvas handling.
*   **Premium Visuals**: High-quality particle physics with glow effects.

---

## 🚀 Installation

Download or clone the repository.
Include the JavaScript file in your website.

```html
<script src="diwali-fireworks.js"></script>
```

---

## 🧑‍💻 Usage Example

```html
<!DOCTYPE html>
<html>
<head>
    <title>Happy Diwali</title>
</head>
<body>

    <h1 style="color: white; text-align: center; margin-top: 20%;">Happy Diwali 🪔</h1>

    <!-- One single file to add fireworks! -->
    <script src="diwali-fireworks.js"></script>

</body>
</html>
```

## 🧠 Concept

Instead of writing fireworks animation separately for every project, this script works as a **centralized JS animation file**. Developers can reuse this file across multiple projects and websites to quickly add **Diwali fireworks celebration effects**.
